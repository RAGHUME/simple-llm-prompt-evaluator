Workspace / How to Run
Quick start and per-page usage notes
Local Setup (One Time)
1.Install Python 3.9+ and Ollama.
2.Create virtual environment and install dependencies with pip install -r requirements.txt.
3.Start Ollama using ollama serve in a separate terminal.
4.Pull at least one model, for example ollama pull phi3:mini.
5.Start backend with python main.py and open http://localhost:8000.
Daily Quick Start
1.Run ollama serve.
2.Run python main.py (or start.bat on Windows).
3.Open the dashboard and pick a model in Evaluate tab.
4.Add your prompt variants and click Evaluate.
How Each Page Works
Evaluate
Create variants, run side-by-side scoring, and inspect best/worst outputs.
Dataset Mode
Upload CSV, batch-evaluate, optimize failures, export CSV or a team bundle ZIP (JSON + chart + PDF).

Matrix

Iterations
Track optimization lineage and score progression over versions.
History
Review runs, PDF per row or full report, and one-click team bundle (ZIP) for sharing.
